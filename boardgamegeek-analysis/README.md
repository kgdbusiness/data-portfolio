# BoardGameGeek Analysis

## Business Questions
- What factors influence board game popularity and ratings?

## Data
- Data sourced from BoardGameGeek datasets.

## Tools Used
- SQL (SQLite)
- Python (pandas)

## Key Questions
- Do more complex games receive higher ratings?
- Does player count impact popularity and ratings?
- Do newer games receive higher ratings than older games?
- Do reimplemented board games receive better ratings than their originals?
